# Academic Review Web Application
Name: Yun Shen Tan

Student Number: 20065126
## Overview
Developing a web application Academic Review with AngularJS, Python, Django REST Framework, MySQL, Amazon Web Services, Mailgun etc.

Academic Review is a platform for users to understand the information of universities by collecting academic reviews that made by students,
lecturer, universities, and companies. The main aim of the project is to create a web application for users to provide their experiences during
college through the reviewing system and gain information about college from the web application.

URL: [academicreview.tk](academicreview.tk)

## Requirements
+ Python 3.6
+ NodeJS 8.9.1

## Installation
Install using pip and npm...

    pip install -r requirements.txt
    npm install
